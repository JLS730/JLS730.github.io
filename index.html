Pawsitively Yours

Pawsitively Yours is dog adoption created using React, Firebase, Sass and Postman to fetch data from the Petfinder API. Users are able to create an account, browse and search dogs, and assign pets as favorites for later viewing. Users are also able to save and manage their personal information that is stored on the firestore database.
